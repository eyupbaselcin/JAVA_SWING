## Getting Started

Welcome to the VS Code Java world. Here is a guideline to help you get started to write Java code in Visual Studio Code.

## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies

Meanwhile, the compiled output files will be generated in the `bin` folder by default.

> If you want to customize the folder structure, open `.vscode/settings.json` and update the related settings there.

## Dependency Management

The `JAVA PROJECTS` view allows you to manage your dependencies. More details can be found [here](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).
## Başlangıç ​​Kılavuzu

VS Code Java dünyasına hoş geldiniz. Visual Studio Code'da Java kodu yazmaya başlamanıza yardımcı olacak bir kılavuz aşağıda verilmiştir.

## Klasör Yapısı

Çalışma alanı varsayılan olarak iki klasör içerir:

- `src`: kaynak dosyaların saklandığı klasör
- `lib`: bağımlılıkların saklandığı klasör

Bu arada, derlenmiş çıktı dosyaları varsayılan olarak `bin` klasöründe oluşturulacaktır.

> Klasör yapısını özelleştirmek istiyorsanız, `.vscode/settings.json` dosyasını açın ve ilgili ayarları orada güncelleyin.

## Bağımlılık Yönetimi

`JAVA PROJELERİ` görünümü, bağımlılıklarınızı yönetmenizi sağlar. Daha fazla ayrıntı [burada](https://github.com/microsoft/vscode-java-dependency#manage-dependencies) bulunabilir.
